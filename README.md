# NLW Return Frontend

Um projeto de um popup para enviar feedback do usuário.

## Tecnologias

- Typescript;
- React;
- Tailwind CSS;

## Funcionalidades

- Acessibilidade;
- Screenshot automático da tela;

## Aprendizados

- Tailwind CSS;
- Detalhes de acessibilidade;
- Componentes HOC;

## Stack utilizada

**Front-end:** React, TailwindCSS

**Back-end:** Node, Express

**Mobile:** React Native
